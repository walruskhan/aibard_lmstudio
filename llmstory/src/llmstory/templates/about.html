What is LLM Story?

LLM Story is a Flask web application that demonstrates modern web development practices using Python. It showcases how to structure a Flask application with proper template inheritance, static file serving, and responsive design.

Key Features

  • Template-based HTML rendering using Jinja2
  • Static file serving for CSS, JavaScript, and images
  • Responsive design with modern CSS
  • Modular project structure
  • Development server with auto-reload

Technology Stack

  • Backend: Flask (Python)
  • Frontend: HTML5, CSS3, JavaScript
  • Package Management: UV
  • Template Engine: Jinja2
